# UK Road Accident Analysis Dashboard

## Project Description

An interactive Power BI dashboard created to analyze road accident data in the United Kingdom.
This project helps understand accident trends, severity, vehicle involvement, road conditions, and time-based patterns to support better road safety decisions.
